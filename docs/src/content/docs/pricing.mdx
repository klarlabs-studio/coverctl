## Today

coverctl's CLI and MCP server are **Apache-2.0 licensed and free**. No
account, no source upload, no SaaS dependency in the agent's reach.

## On the roadmap

A hosted layer for **cross-repo coverage history**, **team dashboards**,
and a **cloud MCP relay** for organisations that prefer not to run local
stdio MCP in CI runners. Additive — does not gate anything that is free
today.

Stage gates and what stays free vs. what becomes paid are documented in
the [monetization decision](https://github.com/felixgeelhaar/coverctl/blob/main/docs/strategy/monetization-decision.md)
strategy file. Stage 1 (sponsorship) is active today; Stage 2 (open
core) triggers on a measurable adoption signal.
